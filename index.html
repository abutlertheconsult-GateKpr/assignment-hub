⬇️ Export Data

Download all assignments, maps, and captains as a JSON backup file. Use this to transfer data to another device or share with a co-admin.

⬆️ Import Data

Load a previously exported JSON file. This will replace all current data.

👤 Personalize & Send

Generate a copy of this app with a specific person's name pre-filled and all current data baked in. Send the downloaded HTML file directly to that person.

The recipient opens the HTML file in their browser and sees their assignments highlighted automatically — no setup needed.

📊 Import from Excel

Upload an Excel file (.xlsx) with your assignments. The file must have a sheet named Friday, Saturday, or Sunday (or all three). Each sheet needs these columns in order: Name, Congregation, Phone, Location, Time.

⚠️ Importing will add rows to existing data. It will not delete what's already there. Use the day tabs to remove duplicates if needed.
